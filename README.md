# sublime-config
Sublime 个人配置备份


### 常用包
* Package Control  (包管理)
* Theme - Flatland (主题)
* Block Corsor Everywhere (vim 块光标)
* BracketHighlighter (括号高亮)
* SideBarEnhancements (侧边栏增强)
* Babel (babel 解析)
* Pretty JSON (JSON格式化)
* HTML-CSS-JS Prettify (代码格式化)
* TypeScript (ts 语法高亮)
* SublimeLinter-eslint (eslint 语法检查)
* GitGutter (Git 修改状态展示)
